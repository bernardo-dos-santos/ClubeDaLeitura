# Clube da Leitura
![](https://i.imgur.com/gGcJ50U.gif)
## Introdução 
Um projeto voltado para a ideia de gerenciamento de um clube de livro ou biblioteca 
## Tecnologias

[![Tecnologias](https://skillicons.dev/icons?i=git,github,cs,dotnet,visualstudio)](https://skillicons.dev)
## Detalhes
o Sistema conta com 5 menus, Sendo: Amigos, Revistas, Caixas, Empréstimos e Reservas

## Funcionalidades
**Menu Amigos** - O usuário tem as opções de Cadastrar, Editar, Excluir e Visualizar amigos

**Menu Revistas** - O usuário tem as opções de Cadastrar, Editar, Excluir e Visualizar revistas

**Menu Caixas** - O usuário tem as opções de Cadastrar, Editar, Excluir e Visualizar caixas

**Menu Empréstimo** - O usuário tem as opções de Registrar um empréstimo, Editar um empréstimo, Visualizar um empréstimo e Devolver um empréstimo, Com adiconal de Visualização de Multas em caso de atraso

**Menu Reservas** - O usuário tem as opções de Cadastrar, Editar, Excluir, Visualizar e Retirar uma reserva

**Tratamento de Entradas** - O sistem oferece uma resposta de erro ao usuário em caso de entrada inválida
 
## Requisitos para uso
.NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto

## Como utilizar
Clone o Repositório
```
git clone https://github.com/bernardo-dos-santos/ClubeDaLeitura
```
Navegue até a pasta raiz da solução
```
cd ClubeDaLeitura
```
Restaure as dependências
```
dotnet restore
```
Navegue até a pasta do projeto
```
cd ClubeDaLeitura.ConsoleApp
```
Execute o projeto
```
dotnet run
```
